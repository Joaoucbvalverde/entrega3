# 📱 Sistema de Biblioteca Mobile - Entrega 3

> Aplicativo móvel desenvolvido com **React Native** e **Expo** em **TypeScript**, focado na gestão de acervos literários com diferentes níveis de acesso.

---

## 📝 Sobre o Projeto
Este aplicativo faz parte da terceira entrega da disciplina. Ele simula um ambiente de biblioteca real, onde usuários com diferentes perfis (Autor, Leitor e Super Admin) podem interagir com o acervo. O projeto foca em uma interface detalhada e funcionalidade de CRUD completa.

### 🛠️ Funcionalidades
- **Login Simulado**: 3 botões para entrada rápida como Autor, Leitor ou Super Admin.
- **Gestão de Acervo**: Listagem detalhada de livros e autores através de Cards.
- **CRUD Completo**: Telas dedicadas para Cadastro, Edição, Detalhes e Exclusão de obras.
- **Interface Moderna**: Uso de ícones, feedbacks visuais e navegação fluida entre 5 telas.

---

## 🚀 Tecnologias Utilizadas

| Tecnologia | Descrição |
| :--- | :--- |
| **React Native** | Framework para desenvolvimento mobile multiplataforma. |
| **Expo (v50+)** | Plataforma para facilitar o desenvolvimento e teste em tempo real. |
| **TypeScript** | Garantia de segurança de tipos e melhor manutenção do código. |
| **React Navigation** | Gerenciamento de rotas e navegação entre as 5 telas. |
| **Expo Vector Icons** | Conjunto de ícones profissionais para a interface. |

---

## 📥 Passo a Passo para Instalação e Teste

Siga estas etapas para rodar o app no seu dispositivo:

### 1. Instalar as Dependências
Abra o terminal na pasta do projeto e execute:
```bash
npm install
```
após isso inicie o projeto no terminal com:
```bash
npx expo start
```
